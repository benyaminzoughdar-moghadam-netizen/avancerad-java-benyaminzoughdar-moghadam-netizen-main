# Fulköpings Köp & Sälj

## Description
Client-server application built with Java and Spring Boot.

## Features
- List ads
- Show ad
- Create ad
- Update price
- Delete ad

## Architecture
Client (Java console) communicates with backend via REST API.

## Endpoints
GET /ads
GET /ads/{id}
POST /ads
PUT /ads/{id}/price
DELETE /ads/{id}
